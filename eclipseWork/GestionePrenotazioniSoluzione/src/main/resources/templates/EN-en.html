The rules for creating a workstation are: -A station is valid for a single day and can only be made if the station is free for that day. -A user can search for workstations indicating the type of workstation desired and the city of interest. -A user can have multiple bookings in progress, but cannot book more than one seat for a particular date. -A request for a seat reservation can be made up to two days before the chosen date
